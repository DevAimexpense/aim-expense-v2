**ensureAllTabsExist** จะ auto-extend column ใน Sheet เก่าให้ user — ไม่ต้องสร้าง sheet ใหม่ ✅

### Note สำคัญเกี่ยวกับ TTLAmount/Subtotal
- **TTLAmount = ก่อน VAT แล้ว** (CostPerUnit × Days × NoOfPPL — ไม่รวม VAT)
- ดังนั้น **ไม่ต้องเพิ่ม column "Subtotal" ใหม่** — ใช้ TTLAmount เป็น "ยอดรวมก่อนภาษี" ใน UI ได้เลย
- แค่ rename label ใน UI/labels เพื่อความชัดเจน → ทำตอน R6

### tRPC payment.router
- `PaymentInputSchema` รับ optional fields ใหม่ทั้ง 5 (documentType, expenseNature, categoryMain, categorySub, requesterName)
- `payment.create` — auto snapshot vendorTaxIdSnapshot + vendorBranchInfo จาก Payee (best-effort) + default `RequesterName = CreatedBy` ถ้าไม่ระบุ
- `payment.update` — รับและอัพเดต tax fields ใหม่; ถ้าเปลี่ยน payeeId → refresh vendor snapshot อัตโนมัติ
- `toPaymentRow()` — return tax fields กลับ frontend

### LINE Handler (handlers.ts)
- `confirmDraftAsync` — populate fields ใหม่จาก OCR:
  - `DocumentType` = `tax_invoice` ถ้า OCR detect, else `receipt`
  - `RequesterName` = "LINE OA"
  - `VendorTaxIdSnapshot` = ocr.vendorTaxId
  - `CategoryMain/Sub` = "" (จะให้ AI suggest ใน R3 ทีหลัง)

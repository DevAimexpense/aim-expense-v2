## 1. การยอมรับเงื่อนไข

การสมัครใช้งาน Aim Expense (เว็บแอป + LINE OA) ถือว่าท่านยอมรับข้อกำหนดและ
[นโยบายความเป็นส่วนตัว](PRIVACY_POLICY_TH.md). ถ้าไม่ยอมรับ กรุณาไม่ใช้บริการ

## 2. คำนิยาม

- **"บริการ"** หมายถึง เว็บแอปและ LINE OA ของ Aim Expense รวมถึง API ที่เปิดให้
- **"องค์กร"** หมายถึง บัญชีองค์กรที่ท่านสร้างบนระบบ
- **"ข้อมูลธุรกิจ"** หมายถึง ใบเสร็จ ใบกำกับภาษี รายการจ่าย ลูกค้า ผู้รับเงิน ฯลฯ
  ที่เก็บใน Google Drive/Sheets ของท่าน
- **"แผนการใช้งาน"** หมายถึง แผน free / basic / pro / business / max / enterprise

## 3. การสมัครและบัญชี

- ท่านต้องมีอายุไม่ต่ำกว่า 18 ปี หรือได้รับความยินยอมจากผู้ปกครองตามกฎหมาย
- ท่านต้องให้ข้อมูลที่ถูกต้องและเป็นปัจจุบัน
- ท่านรับผิดชอบรักษาความปลอดภัยของบัญชี Google / LINE ที่ใช้ login
- ห้ามแบ่งปันบัญชีระหว่างผู้ใช้คนละคน — ใช้ระบบ invite และ permission ภายใน org แทน

## 4. ข้อมูลธุรกิจของท่าน (Data Sovereignty)

- **ท่านเป็นเจ้าของ** ข้อมูลธุรกิจของท่านทั้งหมด เก็บใน Google Drive/Sheets
- เราเข้าถึงข้อมูลผ่าน Google OAuth ตามขอบเขตที่ท่านอนุญาตเท่านั้น
- ท่านสามารถลบบัญชี Aim Expense ได้ตลอดเวลา — ข้อมูลใน Google Sheets/Drive ของท่าน
  ไม่ถูกลบ
- เรามีสิทธิ cache ข้อมูลใน Redis ระยะสั้น (TTL ≤ 60 วินาที) เพื่อ performance
  (ดู [นโยบายความเป็นส่วนตัว](PRIVACY_POLICY_TH.md) ข้อ 3.5)

## 5. ขอบเขตของบริการ

Aim Expense ให้:
- ระบบบันทึกค่าใช้จ่าย (พร้อม OCR ใบเสร็จ)
- ระบบใบเสนอราคา / ใบวางบิล / ใบกำกับภาษี
- รายงานภาษี (ภงด.3, ภงด.53, ภพ.30)
- การทำงานเป็นทีม (role + permissions)
- LINE OA notification

เราไม่:
- ให้คำแนะนำทางภาษีหรือบัญชี — ท่านควรปรึกษาผู้ตรวจสอบบัญชี
- รับประกันความถูกต้องของผล OCR — ท่านต้องตรวจทาน
- รับประกัน uptime 100% — ดู §11

## 6. การใช้งานที่ยอมรับได้

ห้ามใช้บริการเพื่อ:
- กิจกรรมที่ผิดกฎหมาย
- ส่งสแปมหรือเนื้อหาน่ารังเกียจ
- พยายามโจมตี / ทำ reverse engineering / bypass การเก็บเงิน
- เก็บข้อมูลของบุคคลที่สามโดยไม่ได้รับความยินยอม

## 7. แผนการใช้งานและการชำระเงิน

- ราคา / quota แต่ละแผนแสดงใน /pricing
- การคิดเงินรายเดือน / รายปี ผ่าน Stripe (เปิดให้บริการในอนาคต)
- ภาษีมูลค่าเพิ่ม 7% รวมในราคาที่แสดง (เว้นแต่ระบุไว้ชัดเจน)
- ยกเลิกได้ตลอดเวลา — ใช้บริการถึงสิ้นรอบบิลปัจจุบัน
- ไม่มีนโยบายคืนเงินสำหรับช่วงที่ใช้งานแล้ว ยกเว้นกรณีที่เราผิดสัญญา

## 8. การยกเลิกและลบบัญชี

- ท่านลบบัญชีได้ที่ /account/data
- เมื่อลบ:
  - ข้อมูลใน Postgres ของเรา (user / org / subscription metadata) ถูกลบภายใน 30 วัน
  - Audit log ถูกลบหลัง 1 ปีตามกำหนด
  - ข้อมูลใน Google Sheets/Drive ของท่าน **ไม่ถูกลบ** — ท่านคุมเอง
- เราอาจระงับหรือยกเลิกบัญชีของท่านหากพบการละเมิด §6 — โดยจะแจ้งล่วงหน้าหากเป็นไปได้

## 9. ทรัพย์สินทางปัญญา

- ซอฟต์แวร์ Aim Expense, brand, design = ของเรา
- ข้อมูลธุรกิจ = ของท่าน
- ข้อเสนอแนะ (feedback) ที่ท่านส่งให้เรา → เราใช้ได้โดยไม่ต้องจ่ายเงิน (non-exclusive)

## 10. การรับประกันและความรับผิด (Limitation of Liability)

- บริการให้ตามสภาพ "AS IS" ไม่มีการรับประกันที่ชัดแจ้งหรือโดยปริยาย
- เราไม่รับผิดต่อ:
  - ความเสียหายทางอ้อม / consequential damages
  - การสูญหายของข้อมูลที่อยู่ใน Google Drive ของท่าน (Google เป็นผู้ให้บริการ)
  - ความเสียหายจากการตัดสินใจทางภาษี / บัญชีที่อ้างอิงผลของเรา
- ความรับผิดสูงสุดของเราจำกัดที่ **ค่าบริการที่ท่านชำระให้เราใน 12 เดือนล่าสุด**
- ข้อจำกัดข้างต้นไม่ใช้กับ: ความเสียหายจากการกระทำโดยจงใจหรือประมาทเลินเล่ออย่างร้ายแรง,
  ความรับผิดที่กฎหมายไม่อนุญาตให้จำกัด
